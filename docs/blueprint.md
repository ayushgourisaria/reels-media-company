# **App Name**: ReelsPulse

## Core Features:

- Content Blueprint Showcase: Interactive section detailing the structured process of Instagram channel creation and automation workflows.
- AI Reel Caption Tool: An AI-powered tool that allows users to input video concepts and generate optimized Instagram captions and trending hashtag sets.
- Inquiry Management: A streamlined form for potential clients to book consultations, with data stored in a lightweight database.
- Lead Dashboard: A protected administrative view to track incoming business requests for owner Ayush Gourisaria.
- Automated Content Timeline: A visual timeline displaying how the semi-automation process speeds up content delivery from idea to upload.

## Style Guidelines:

- Primary color: Electric Cobalt (#446DF5) reflecting the dynamic energy of viral video media.
- Background color: Deep Charcoal (#111317), providing a high-contrast cinematic backdrop that highlights video content.
- Accent color: Cyan Frost (#40CCFF) for call-to-action buttons and process highlights.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and systematic appearance.
- Body font: 'Inter' (sans-serif) for high readability across dense workflow descriptions and contact details.
- Mobile-first vertical scroll design that mimics the UX of the Instagram Reels feed to ensure industry relevance.
- Sleek 'shimmer' and 'fade-in' transitions to represent the automated flow and high production value of the reels.
